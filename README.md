# Telegram Gateway Service

This is a production-grade **Telegram API Gateway** built with FastAPI and Redis.

## Modules

- [**Telegram Gateway**](./telegram_gateway): Handles webhook reception, session management, rate limiting, and routing.

## Getting Started

Navigate to the gateway directory:
```bash
cd telegram_gateway
```
